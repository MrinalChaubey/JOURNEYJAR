# 🌍 JourneyJar

JourneyJar is a full-stack travel journaling web application that helps users capture, organize, and relive their travel experiences. Users can create travel stories, upload photos, mark favorite memories, search through entries, and filter stories based on travel dates.

## 🚀 Live Demo

🌐 https://journeyjar-eta.vercel.app

## ✨ Features

* 🔐 Secure User Authentication (Sign Up & Login)
* 📝 Create, Edit, and Delete Travel Stories
* 📸 Upload and Manage Travel Images
* ⭐ Mark Stories as Favorites
* 🔍 Search Stories by Title, Content, or Location
* 📅 Filter Stories by Travel Date
* 📱 Responsive and User-Friendly Interface
* ☁️ MongoDB Atlas Cloud Database Integration

## 🛠️ Tech Stack

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/MrinalChaubey/JOURNEYJAR.git
cd JOURNEYJAR
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside the backend directory:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

Run the backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
